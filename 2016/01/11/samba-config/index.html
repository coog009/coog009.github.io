Samba快速配置

By Jun Zhao

发表于 2016-01-11

文章目录

  1. 安装samba

    sudo apt-get install samba system-config-samba samba-client samba-common

  2. 打开/etc/samba/smb.con，添加如下内容

    [aliyun]
path = /root
valid users = root
writeable = yes
printable = no
create mask = 0777
directory mask = 0777

  3. 重启samba服务

    service sambd restart
